# 第一章：基礎理論

## 1.1 紫微斗數概述

紫微斗數是中國傳統命理學中最精密的推命術之一，源自唐末宋初陳希夷（陳摶）所創。其特色在於以「宮位」與「星曜」的組合來論斷人生各方面。

### 核心原則（倪師觀點）
1. **先天為命，後天為運** - 命宮看先天，身宮看後天
2. **星曜為體，四化為用** - 星曜決定本質，四化決定變化
3. **三方四正為核心** - 任何宮位都要看其三方四正的整體配置
4. **男女有別** - 同樣的星曜組合，男女命意義不同

## 1.2 十二宮位系統

### 宮位定義與含義

### 身宮的重要性（倪師特別強調）

身宮代表後天發展方向，只會落在以下六個宮位：

| 身宮所在 | 後天發展方向 |
|----------|--------------|
| 命宮 | 身命同宮，先後天一致，個性強烈不易改變 |
| 財帛宮 | 後天重視財富，會去私人企業或自己創業 |
| 官祿宮 | 後天重視事業，適合做官或從政 |
| 遷移宮 | 後天會去外地發展 |
| 夫妻宮 | 後天被婚姻影響很大 |
| 福德宮 | 後天重視精神生活 |

### 天馬星詳解

**基本特性**：
- 只落在「四馬地」：寅、巳、申、亥四宮
- 性格急躁好動、閒不住、開朗熱心
- 活動力強、體力充沛、不安現狀
- 為勞碌命，愈動愈有財
- 天馬落在哪個宮位，就為那個宮位的事情奔波

**吉祥組合**：
| 組合名稱 | 條件 | 效果 |
|---------|------|------|
| 祿馬交馳 | 與祿存同宮 | 發大財、貴人多 |
| 扶轝馬 | 與紫微或天府同宮 | 官運亨通 |
| 財馬 | 與太陰、武曲、天府、化祿同宮 | 財源廣進 |
| 雄馬 | 與太陽同宮 | 早年成功 |
| 雌馬 | 與太陰同宮 | 財源穩定 |

**凶象組合**：
| 組合名稱 | 條件 | 效果 |
|---------|------|------|
| 死馬 | 與地空同宮 | 做事無力、奔波無成 |
| 折足馬 | 與陀羅同宮 | 勞而無功 |
| 戰馬 | 與火星、鈴星同宮 | 蠻幹、開拓但辛苦 |
| 是非馬 | 與化忌同宮 | 是非困擾多 |
| 空亡馬 | 與地空、地劫同宮 | 一生奔波無成 |

## 1.4 四化星系統

### 四化基本含義

| 四化 | 性質 | 代表 | 最喜落宮 |
|------|------|------|----------|
| 化祿 | 財祿 | 財、多得、守財 | 財帛宮、命宮 |
| 化權 | 權力 | 掌控、官印、性剛 | 官祿宮（最佳） |
| 化科 | 科名 | 名氣、專業、考試 | 命宮、官祿宮 |
| 化忌 | 劫殺 | 破耗、停滯、官司 | 任何宮位皆不喜 |

### 四化落宮效果（倪師重點）

**化祿**：
- 在財帛宮：財源廣進
- 在官祿宮：事業順利
- 在父母宮：祖上有產業
- 遇天馬：發大財

**化權**：
- 在官祿宮：最佳，主事業有成
- 在財帛宮：自己做老闆
- 在命宮：個性剛強，有領導力

**化科**：
- 在命宮：才華橫溢
- 在官祿宮：適合考公職
- 遇文昌：金榜題名

**化忌**：
- 在命宮對宮（遷移宮沖命）：大凶，半空折翅
- 在子女宮：無兒子
- 在福德宮：一輩子不順，入僧道可解
- 在僕役宮：朋友變仇人

# 第二章：看盤方法論

## 2.1 倪師看盤步驟

### 第一步：定盤
1. 確認出生年月日時是否正確
2. 確認是用陽曆還是農曆
3. 排出命盤

### 第二步：看四化
1. 先找出化祿、化權、化科、化忌落在哪些宮位
2. 四化是整個命盤的「動力來源」

### 第三步：看命宮三方四正
1. 命宮的星曜組合
2. 財帛宮、官祿宮、遷移宮的星曜
3. 整體格局是吉是凶

### 第四步：看身宮
1. 身宮落在哪個宮位
2. 身宮的星曜組合
3. 判斷後天發展方向

### 第五步：逐宮分析
1. 根據詢問重點，分析相關宮位
2. 注意該宮位的三方四正
3. 結合四化影響

### 第六步：綜合論斷
1. 整合所有資訊
2. 判斷吉凶輕重
3. 給出具體建議

## 2.2 三方四正分析法

### 核心概念

三方四正是紫微斗數看盤的核心方法。以某宮為起點：
- **對宮**：正對面的宮位
- **三方**：順時針第5宮 + 逆時針第5宮 + 對宮
- **四正**：三方 + 本宮

### 各宮三方四正

```
命宮三方四正 = 命宮 + 財帛 + 官祿 + 遷移
  → 看一個人的核心命格

夫妻宮三方四正 = 夫妻 + 遷移 + 官祿 + 福德
  → 看婚姻感情

財帛宮三方四正 = 財帛 + 命宮 + 遷移 + 官祿
  → 看財運

官祿宮三方四正 = 官祿 + 命宮 + 財帛 + 遷移
  → 看事業
```

### 分析原則
1. 吉星多於凶星 → 該宮位事項順利
2. 凶星多於吉星 → 該宮位事項多阻礙
3. 四化落入三方四正 → 加強或削弱該宮位
4. 煞星會入 → 需特別注意

## 2.3 星曜亮度（廟旺陷）

### 亮度等級

| 等級 | 意義 | 效果 |
|------|------|------|
| 廟 | 最強 | 優點放大，缺點縮小 |
| 旺 | 次強 | 力量較強 |
| 得/利 | 中等 | 正常發揮 |
| 平 | 普通 | 中性 |
| 陷 | 最弱 | 優點不顯，缺點明顯 |

### 重要注意（倪師觀點）
- **入廟不一定吉，落陷不一定凶**
- 要看整體組合，不能只看單星亮度
- 太陽、太陰要看出生時辰（日生人太陽強，夜生人太陰強）
- 天府、天相的廟旺不是最重要，重點看組合

| 天府 | 文官、教育 |
| 天相 | 秘書、助理、輔佐 |
| 七殺/破軍/貪狼 | 軍警、武職 |
| 機月同梁格 | 公務員（吏） |
| 武官星+化科 | 自由業（醫師、律師、會計師） |

### 老闆命判斷
- 化權+祿存在財帛宮 → 必定自己做老闆
- 權祿巡逢格 → 自己創業
- 武曲+化權 → 財務掌控強

## 5.2 財運分析

### 看盤重點
1. **財帛宮** - 財運主宮
2. **命宮三方四正** - 有無財星
3. **祿存、化祿、天馬位置**

### 財運等級

| 組合 | 財運等級 |
|------|----------|
| 祿馬交馳（祿存+天馬） | 大富 |
| 化祿+天馬 | 大富 |
| 武曲廟+化祿 | 財星王，大富 |
| 貪狼廟+火/鈴星 | 武貴發財 |
| 財星入財帛宮 | 財運佳 |
| 化忌入財帛宮 | 財運阻礙 |
| 地劫/天空入財帛 | 空耗 |

### 正財 vs 橫財
- **正財星**：太陰、天機、文昌、文曲
- **橫財星**：太陽、貪狼、武曲
- 正財星入財帛 → 適合穩定收入
- 橫財星入財帛 → 可嘗試投資創業

## 5.3 婚姻分析

### 看盤重點
1. **夫妻宮** - 婚姻主宮
2. **福德宮** - 女命最重要
3. **紅鸞、天喜位置**

### 男女差異
- **男命**：先看夫妻宮，次看福德宮
- **女命**：先看福德宮，次看夫妻宮（倪師特別強調）

### 婚姻吉凶判斷

| 組合 | 效果 |
|------|------|
| 紅鸞入命 | 男招美妻，女有貴夫 |
| 左輔/右弼單守夫妻宮 | 二婚 |
| 破軍入夫妻宮/福德宮 | 婚姻不好 |
| 桃花星入夫妻宮 | 感情豐富 |
| 化忌入夫妻宮 | 婚姻有阻礙 |
| 紅鸞+天喜入子女宮 | 即使化忌也必結婚 |

### 破解方法（倪師傳授）
- 婚姻凶 → 找二婚對象 或 年齡差7歲以上

## 5.4 健康分析

### 看盤重點
1. **疾厄宮** - 健康主宮
2. **疾厄宮地支** - 對應身體部位
3. **煞星位置**

### 地支與身體對應

| 地支 | 對應臟腑 |
|------|----------|
| 子 | 膽 |
| 丑 | 肝 |
| 寅 | 肺、右腳 |
| 卯 | 大腸 |
| 辰 | 胃 |
| 巳 | 脾 |
| 午 | 心 |
| 未 | 小腸 |
| 申 | 膀胱 |
| 酉 | 腎 |
| 戌 | 心包 |
| 亥 | 三焦 |

### 判斷原則
- 煞星多在哪個宮 → 該宮對應的身體部位需注意
- 化忌入疾厄宮 → 健康需特別注意
- 廉貞+七殺/破軍 → 有血光之災風險

## 5.5 子女分析

### 代表兒子的星
紫微、天相、貪狼、太陽、天府、武曲、天同、文昌

### 代表兩個兒子的星
破軍、廉貞、七殺

### 子女緣判斷
- 化忌入子女宮 → 無兒子
- 子女宮空宮或煞星多 → 子女緣薄
- 吉星入子女宮 → 子女有成就

# 第六章：綜合論斷方法

## 6.1 全盤分析流程

```
步驟一：建立基本框架
├── 1. 確認命宮主星 → 定個性基調
├── 2. 確認身宮位置 → 定後天發展
└── 3. 確認四化落點 → 定變化動力

步驟二：核心格局判斷
├── 1. 是否成吉格 → 富貴程度
├── 2. 是否有凶格 → 需注意事項
└── 3. 整體吉凶比例

步驟三：三方四正分析
├── 1. 命宮三方四正 → 核心命格
├── 2. 相關宮位三方四正 → 專題分析
└── 3. 吉凶星分布

步驟四：專題深入
├── 1. 事業運 → 官祿宮系統
├── 2. 財運 → 財帛宮系統
├── 3. 婚姻 → 夫妻宮+福德宮
├── 4. 健康 → 疾厄宮系統
└── 5. 其他 → 相關宮位

步驟五：綜合論斷
├── 1. 整合所有資訊
├── 2. 分辨輕重緩急
├── 3. 給出具體建議
└── 4. 說明破解方法
```

## 6.2 論斷優先級

### 第一優先：看有無大凶格
- 半空折翅（化忌沖命）
- 天羅地網
- 廉貞貪狼落陷
- 日月反背+化忌沖命

### 第二優先：看四化落點
- 化忌在哪個宮位 → 該宮位事項需注意
- 化祿化權在哪 → 該宮位事項順利

### 第三優先：看命宮主星組合
- 有無輔弼來會
- 是否落陷
- 與其他星曜組合

### 第四優先：看專題宮位
- 根據詢問重點分析

## 6.3 論斷表達方式

### 正確的表達
```
「命宮紫微入廟，有左輔右弼來會，為大貴之格。
三方四正財帛宮有武曲化祿，官祿宮有化權，
整體命格偏向富貴，適合擔任領導職位。
後天身宮在官祿宮，代表會往仕途發展。」
```

### 錯誤的表達（條列式）
```
「命宮：紫微入廟
左輔：有
右弼：有
財帛宮：武曲化祿
...」
```

## 6.4 破解與建議

### 常見破解方法

| 問題 | 破解方法 |
|------|----------|
| 婚姻凶 | 找二婚對象；年齡差7歲以上 |
| 化忌沖命+日月反背 | 過繼給別人養 |
| 化忌入福德宮 | 入僧道可解 |
| 七殺臨身宮 | 從事需要魄力的行業 |
| 桃花過重 | 從事與異性相關行業（服務業） |

## 7.3 分析品質標準

### 必須做到
1. **整體觀** - 不能只看單星，要看組合
2. **三方四正** - 任何宮位都要看其三方四正
3. **男女區別** - 同樣組合，男女解讀不同
4. **敘事表達** - 不能只是條列，要有連貫論述
5. **實用建議** - 要給出可操作的建議

### 必須避免
1. **單星論斷** - 不能說「紫微入命就是帝王命」
2. **忽略亮度** - 廟旺陷會影響效果
3. **忽略組合** - 紫微單守 vs 紫微+輔弼完全不同
4. **男女混淆** - 七殺入命，男女意義不同
5. **只報凶不給解** - 說了凶格要給破解方法
